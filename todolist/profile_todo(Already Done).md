# 个人空间（Profile）功能升级 ── 想要实现的 TODO List
### 1. 核心成长与成就激励
- **一个“连续学习天数”的成就计数勋章**。当用户每天登录网站并点击签到时，这个数字会累加，以此正向激励大家每天都愿意回来看一眼。
    - **短期保护**：连续签到 7 天以上，漏签后不立刻归零，显示“3天后归零”，3天内签到立刻重燃。
    - **长期保护**：连续签到 45 天以上，漏签后不立刻归零，显示“7天后归零”，7天内签到立刻重燃。
    - **回归奖励**：若缓冲期过完彻底归零，用户再次回来签到时，触发老友奖励机制（预留接口，回头再加）。
- **资料填写完整度**：克制的 MD3 胶囊进度条（如 2/7 已完成）。按昵称、专业、年级、QQ、微信、肖像描述、MBTI 人格（给出十六个选项INTJ、INTP、ENTJ、ENTP、INFJ、INFP、ENFJ、ENFP、ISTJ、ISFJ、ESTJ、ESFJ、ISTP、ISFP、ESTP、ESFP的选择框） 7 项因子各占等额权重计算。
- **一个“资料填写完整度”的指示条**。用一个干净、克制的 MD3 胶囊进度条，直观告诉用户他的名片还差哪些信息没补全（比如：已完成13%）。

### 2. 社交名片与关注系统

- **一个“关注”和“粉丝”的计数与列表功能**。用户之间可以相互点击关注，在个人主页能看到自己关注了多少人、有多少个粉丝，点击能看到具体的同学列表。

### 3. 高级感的功能控制列表（MD3 List Items）
- **把目前零散、松散的隐私开关和退出按钮，整合成高聚合度、高级感的交互功能列表**。
- 每一个功能条都要有**左侧语义化图标**、**上方加粗标题**、**下方灰色解释小字**，以及**最右侧的右箭头（`chevron_right`）**。
- 列表里需要包含以下几个高定交互条：
    - **【编辑资料】**：现在的版本是点击打开弹窗更新基础信息。我们直接摒弃，采用全新设计语言打开新的edit界面。
			基本资料：昵称、专业、年级、QQ、微信号、抖音号
			个性资料：个人肖像描述（80字以内）、人格选择（采用MBTI人格选项）
	- **【隐私与安全】**：
		- 隐私：管理是否公开信息、是否允许被匹配。
		- 安全：登录保护（只允许白名单设备登录等）
	- **【管理我的数据】**：点击查看数据概览、下载个人信息、注销账号。

每次点击后，进入诸如profile/edit这样的二级子网页。

### 4. 个人主页系统升级

- 个人主页方面，用户拥有三种不同的视角：
    1. **【公开主页】**：这是大家每个人点击别人的头像时，看到的干净利落的对外名片。
    2. **【在线直接编辑视角】**：我自己看自己的主页时，不需要进死板的弹窗，点击文字就能直接修改。
    3. **【第三方视角预览功能】**：我自己看主页时，可以一键切换到“**预览他人看我**”的视角，用来肉眼检查我的隐私隐藏得够不够彻底。
### 5. 开发者连接通道

- **我想要一个“问题反馈”功能**。用户如果在用网站时血压飙升或者遇到 Bug，点击这个列表项，能直接呼出反馈渠道联系到开发者。
