## 页面设计

### 登录/注册页
- 居中卡片，max-width 400px
- Tab 切换"登录"/"注册"
- 登录：用户名 + 密码
- 注册：用户名 + 密码 + 昵称 + 专业 + 年级
- 错误提示红色文字

### 课程列表页
- 标题"课程列表" + "创建课程"按钮
- 课程卡片：课程名、教师、学期、人数
- 点击卡片 → 进入课程空间
- "加入课程"按钮

### 课程空间页（论坛）
- 顶部：课程信息（名、教师、描述）
- 左栏：帖子列表 + 发帖按钮
- 右栏：成员列表
- 帖子卡片：标题、作者、时间、评论数
- 点击帖子展开评论

### 个人中心
- 已登录：头像占位、昵称、用户名、专业、年级、退出
- 未登录：登录/注册表单

## 组件规范

### 卡片
- 白色背景，1px solid #e5e7eb 边框
- 圆角 8px，内边距 24px
- 悬停：translateY(-2px) + 阴影

### 按钮
- Primary: filled, 无渐变
- Secondary: outlined
- 圆角 8px

### 颜色
| Token | 值 |
|-------|-----|
| --text-primary | #111827 |
| --text-secondary | #6b7280 |
