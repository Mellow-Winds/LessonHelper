## 当前项目状态

### 后端 — 17个API端点全部可用
| 模块 | 端点 | 认证 |
|------|------|------|
| 认证 | register, verify-email, login, resend-code | 否 |
| 认证 | GET me, PUT me | 是 |
| 课程 | GET 列表, GET 详情, GET 成员 | 否 |
| 课程 | POST 创建, POST 加入 | 是 |
| 帖子 | GET 列表, POST 发帖 | 发帖需认证 |
| 评论 | GET 列表, POST 发评论 | 发评论需认证 |
| 用户 | GET 公开信息, GET 课程 | 否 |

### 前端 — 3个页面
1. **课程列表页** — 浏览课程、创建课程、选课人数
2. **课程空间页（论坛）** — 帖子列表、发帖、评论、成员侧栏
3. **个人中心** — 登录/注册/验证/编辑资料/退出

### 邮件配置
- 开发模式：不配 Resend API Key，验证码在响应中返回
- 生产模式：设置 `RESEND_API_KEY` 环境变量即可发送真实邮件
