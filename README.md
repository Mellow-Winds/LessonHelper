# 课搭子 — 同课程学习互助平台

面向大学生的同课程学习互助平台，以南京大学蓝鲸文化为品牌背景。

## 作品目标

**课搭子**致力于解决大学生"找课友难、共享资料散、约自习低效"三大痛点：

1. **课程聚合** — 通过课表导入自动归并大课，跨班级共享论坛、资料、搭子帖，打破行政班壁垒
2. **学习互助** — 分类资料上传/下载/评分、论坛式讨论区（楼中楼评论 + 图片回复 + @提及）
3. **社交连接** — 交友广场（兴趣匹配 + 双向确认交换联系方式）、自习邀约、关注动态
4. **效率工具** — 番茄钟、运势、决策助手、盲盒、AI 搜索等百宝箱功能
5. **隐私优先** — 细粒度隐私控制、预览"他人看我"模式、肖像文字描述替代真实照片

## 使用指南

### 环境要求

- **Node.js** ≥ 18.x
- **npm** ≥ 9.x
- 可选：Resend API Key（邮件验证码服务）

### 部署指南

#### 1. 克隆仓库

```bash
git clone git@github.com:Mellow-Winds/LessonHelper.git
cd LessonHelper
```

#### 2. 安装依赖

```bash
npm install
```

#### 3. 配置环境变量

创建 `.env` 文件（可选，开发模式下不配置邮件也能运行）：

```env
# Resend API Key（用于发送验证码邮件）
RESEND_API_KEY=re_xxxxxx

# JWT 密钥（不设置则使用默认值）
JWT_SECRET=your-secret-key
```

#### 4. 启动服务

```bash
# 开发模式
npm run dev

# 生产模式
NODE_ENV=production npm start
```

服务器默认监听 `http://localhost:3000`。

#### 5. 验证运行

```bash
curl http://localhost:3000/
# 应返回 HTML 页面
```

### 使用指南

#### 注册与登录

1. 访问 `http://localhost:3000`，点击侧边栏"登录/注册"
2. 输入邮箱 → 点击"发送验证码" → 查收邮件（开发模式下验证码显示在页面顶部）
3. 输入验证码 → 设置密码 → 完成注册
4. 登录后完善个人资料（头像/昵称/专业/年级等）

#### 导入课表

1. 进入"我的课程"→ 点击"导入课表"
2. 上传南京大学教务系统导出的 XLSX 课表文件
3. 系统自动解析课程并归并到大课空间

#### 课程空间

- **论坛**：发帖/评论（支持楼中楼 + 图片 + @提及）
- **资料**：上传课件/笔记/作业/真题，按分类和章节筛选
- **交友**（已选课用户）：发布搭子帖，寻找同课伙伴

#### 发现模块

- 浏览全局帖子流，点击卡片查看详情
- 发帖支持富文本编辑器 + 嵌入功能卡片
- 评论区支持内联回复 + 图片上传 + 30 秒冷却

#### 自习邀约

- 进入发现页 → 自习邀约 Tab
- 发布邀约：设定时间/地点/人数上限
- 浏览邀约列表，点击加入
- 在"我的邀约"中管理发起和参与

#### 交友广场

- 进入发现页 → 交友广场 Tab
- 选择分类（考研/考公/考证/项目组队等）发布需求帖
- 感兴趣 → 双方确认 → 交换联系方式
- 帖子 7 天自动过期

#### 消息通知

- 顶部铃铛图标显示未读数量
- 点击通知卡片直接跳转到对应内容（评论精确定位）
- 支持批量管理和全部标记已读

#### 百宝箱

- 番茄钟：25 分钟专注 + 5 分钟休息
- 今日运势：每日随机运势卡片
- 决策助手：帮你做选择
- 盲盒：随机惊喜内容

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Express 4.x + SQLite (sql.js WASM) |
| 认证 | JWT (jsonwebtoken) + bcryptjs |
| 邮件 | Resend API |
| 文件上传 | Multer（磁盘存储） |
| 图片处理 | Sharp（后端） + Canvas（前端三级压缩） |
| 课表解析 | xlsx (SheetJS) |
| 前端 | 原生 JavaScript SPA + ES6 Modules |
| 设计系统 | Material Design 3（自定义 CSS Token） |
| Markdown | markdown-it |
| 图标 | Material Icons Round + Remix Icon |
| 测试 | node:test（17 个测试文件，49 个测试用例） |
