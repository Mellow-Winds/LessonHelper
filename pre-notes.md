# 请阅读并同意使用须知
1. 为防止恶意注册、刷课，本功能仅供用户本人使用。每用户每学期仅可添加至多**50门**课程。
导入/添加课程按学期阶段限流，规则如下：

   - 第一学期（8.15-1.1）：仅可导入1次
   - 寒假（1.2-2.14）：无法导入课程
   - 第二学期（2.15-6.14）：仅可导入1次
   - 暑期（6.15-8.14）：仅可导入1次

**为了不影响您进行补退选操作导致无法正常使用**，我们支持通过搜索选择已有课程加入，也可自主退出已添加课程。
## 请明确以上须知，并谨慎导入您的课表。
2. 请确保按照要求上传的 xlsx 文件为教务系统导出的原始格式，否则会导致解析失败。
3. 导入的课程数据将与你的账号绑定，其他用户无法查看你的课程列表。
4. 课程空间为共享区域，同一课程的所有选课用户均可参与讨论。
